# Tech News Aggregator

A clean, minimal tech news aggregator frontend built with React, Vite, TypeScript, and Redux Toolkit.

## Setup

1.  Navigate to the project directory:
    ```bash
    cd frontend
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

## Features

-   **Feed**: Displays tech news stories with AI-generated summaries.
-   **Redux Toolkit**: Manages application state.
-   **TypeScript**: Fully typed codebase.
-   **Responsive Design**: Optimized for desktop but works on mobile.

## Configuration

-   The API base URL can be configured using the `VITE_API_BASE_URL` environment variable. Default is `/api`.
